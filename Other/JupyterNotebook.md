# Jupyter Notebook

## Installation and Basic Usage

### As Local Editor

Installation

```sh
python3 -m pip install --upgrade pip
python3 -m pip install jupyter
```

Usage

```sh
# open local web server
jupyter notebook
ipython notebook

# CLI
ipython
```
